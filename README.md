## 📋 Overview
ShopSmart is an intelligent e-commerce platform that leverages AI algorithms to provide personalized product recommendations based on user behavior and preferences.

## ✨ Features

### 🔐 User Management
- User registration & authentication
- Persistent login with LocalStorage
- Personalized user profiles

### 🎯 Smart Recommendations
- **Collaborative Filtering** - Based on user interactions
- **Content-Based Filtering** - Category and product similarity
- Real-time recommendation updates
- "Refresh Recommendations" feature

### 🛒 Product Features
- 80+ products across 8 categories
- 10 products per category
- Product search & filtering
- Add to favorites functionality
- Product view tracking

### 🎨 UI/UX
- Modern gradient design
- Responsive grid layout
- Toast notifications
- Loading animations
- Modal dialogs for auth


## 🚀 How It Works

### Recommendation Algorithm
1. Tracks user product views
2. Analyzes category preferences
3. Finds similar products from preferred categories
4. Excludes already viewed items
5. Returns top 8 personalized recommendations

### Data Flow
User Login → Browse Products → View Items →
Update Interactions → Refresh Recommendations →
Get Personalized Suggestions

## 📦 Installation

# Clone the repository
git clone https://github.com/yourusername/ShopSmart-AI-Recommendation-Engine.git

# Navigate to project directory
cd ShopSmart-AI-Recommendation-Engine

# Open in browser
open ecommerce_demo.html
🎮 Usage Guide
Register a new account or Login with existing credentials

Browse products from 8 different categories

Click on any product to view details (triggers recommendations)

Add products to favorites with ❤️ button

View personalized recommendations in the "Personalized For You" section

Use filters to search by name or category

Refresh recommendations anytime with the refresh button
